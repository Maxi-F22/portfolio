# Portfolio

This is my personal portfolio website showcasing a selection of my projects from my Bachelor's and Master's studies in Media Informatics.

You can view the live version here:

[Portfolio](https://maxi-f22.github.io/portfolio/)

## About the Project

This portfolio was built using **Next.js** and styled with **Tailwind CSS**. It was bootstrapped using `create-next-app`.

The goal of this site is to present my work in a clean and structured way, highlighting both technical and creative projects from areas such as game development, web development, graphics programming, and interactive systems.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- TypeScript

## Features

- Responsive design for different screen sizes
- Project-based structure for easy navigation
- Clean and minimal UI focused on content
- Built with performance and simplicity in mind
